# Mobile Music App (Sequencer)

A mobile-friendly music creation application (sequencer) built with Godot Engine.

## Features
- **Dynamic Track Generation**: Create multiple instrumental tracks dynamically with individual volume controls.
- **Advanced Piano Roll Editor**: A 16-step sequencer with an adjustable 17-column grid and drag-and-draw note input.
- **Polyphonic Audio Engine**: Capable of playing multiple pitches simultaneously using dynamic `AudioStreamPlayer` instances.
- **Seamless Audio Export**: Mathematically precise looping export mechanism capturing exactly 16 steps using `AudioEffectRecord`.
- **Cross-Platform Compatibility**: Fully supports Android devices and WebAssembly (HTML5) deployments.
- **Premium UI/UX**: Dark mode aesthetic with custom vector styling (`StyleBoxFlat`) and micro-animations for an intuitive experience.

## Getting Started

1. Clone the repository.
2. Open the project in Godot Engine.
3. Press "Play" to test the app, or export it to your desired platform.
